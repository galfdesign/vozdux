## Приложение «Качество воздуха · Россия»

Веб‑приложение на Vite + TypeScript с картой России (Leaflet).  
При клике по точке на карте загружаются онлайн‑данные о качестве воздуха и отображаются в карточке справа.

### Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запустите dev‑сервер:

```bash
npm run dev
```

3. Откройте указанный в консоли адрес (обычно `http://localhost:5173/`) в браузере.

### Как это работает

- Слева отображается интерактивная карта (OpenStreetMap + Leaflet), приблизительно центрированная на Россию.
- При клике по любой точке:
  - ставится маркер,
  - отправляется запрос к сервисам качества воздуха (см. ниже),
  - справа в карточке появляются найденные параметры загрязнения (PM2.5, PM10, NO₂ и т.д.).
- Если для точки нет данных, показывается аккуратное сообщение об их отсутствии.

### Источники данных

- **OpenWeather Air Pollution API** — основной источник, если вы укажете API‑ключ.
  - Получите бесплатный ключ на сайте OpenWeather.
  - Создайте в корне проекта файл `.env` и добавьте строку:
    ```bash
    VITE_OPENWEATHER_API_KEY=ВАШ_КЛЮЧ
    ```
  - После этого перезапустите dev‑сервер.
- **Open‑Meteo Air Quality API** — запасной источник, не требующий ключа.
  - Используется автоматически, если ключ OpenWeather не указан или сервис временно недоступен.

> Важно: охват и точность сильно зависят от самих сервисов (модели прогноза, плотность станций наблюдения и т.п.). Для конкретных городов России обычно данные есть хотя бы в одном из источников, но возможны точки, для которых сведений нет.
